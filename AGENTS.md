# Инструкции для агентов

Перед началом задачи **прочитай полностью** [docs/new-agents.md](docs/new-agents.md).

- Следуй разделу **Rules** в этом файле.
- Учитывай все секции файла при выполнении задач.

Единственный источник правды по gotchas и конвенциям проекта — [docs/new-agents.md](docs/new-agents.md).

## Дополнительно

Если в сообщении пользователя есть ссылка на файл из `docs/prompts/*.md` (например, `@docs/prompts/...`) и нет явного указания на другой режим, трактуй это как команду **немедленно выполнить инструкции из этого промпта**, а не как запрос «проанализировать файл».
