# Overeact Code MVP Update — 2026-04-28

This release completes the first end-to-end MVP cycle for the Overeact Code VS Code extension.

- Added a side composer workflow to submit tasks, review plan/progress, inspect tool activity, and receive final summaries.
- Improved safety and privacy defaults: permission prompts for risky actions, patch diff review before apply, and stronger secret redaction in logs/summaries.
- Added retrieval-first context handling, verification/retry feedback loops, and terminal/tool execution tracking for more reliable local agent runs.
- Documented a local dogfooding smoke scenario with an Extension Development Host launch check and key open MVP questions for next iteration planning.
